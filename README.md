# Web Scraping/Crawling Studies 

Repository created to host personal research on web scraping/crawling.

## Overview

In this repository, you will find:

- **Python Scripts**: Python scripts for scraping data from various websites using libraries like BeautifulSoup, Scrapy, etc.
- **Examples**: Examples of scraping tasks for different websites.

## Getting Started

Most of the studies were done using Jupyter Notebooks.
